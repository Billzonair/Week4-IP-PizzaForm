# Project-Name : Must It Pizza
This is my fourth IP and it is an online form to order Pizzas .


## Technologies Used

- HTML & CSS,Bootstrap & Javascript,JQuery


##### Setup Instructions and Installation

step 1 : open a terminal
step 2 : use the code "git clone https://github.com/Billzonair/Week4-IP-PizzaForm.git
step 3 : use the terminal to navigate to the root folder "cd week4-independent-project"
step 4 : once inside the folder launch the file "index.html" using a web browser
